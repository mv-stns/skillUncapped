Berechtigungen erforderlich

Damit alles reibungslos läuft, müssen wir die CORS-Einstellungen aktivieren. Befolgen Sie aus diesem einfachen Grund diese Schritte:

  1. Verwenden Sie einen modernen Browser wie Chrome.
  2. Installieren Sie eine CORS Allow-Erweiterung wie: https://chrome.google.com/webstore/detail/allow-cors-access-control/lhobafahddgcelffkeicbaginigeejlf
  3. Fügen Sie diese Website der Whitelist der Erweiterung hinzu.
  4. Fettisch!
